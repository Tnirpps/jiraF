# Сьют 10: Ссылки - извлечение, нормализация, отображение

---

## TC-LNK-001: URL с https:// извлекается из сообщения

**Предусловия:**
- Активная сессия, сообщение содержит "Смотри https://example.com/issue"

**Шаги:**
1. Написать сообщение с HTTPS-ссылкой
2. Отправить `/create_task`

**Ожидаемый результат:** URL попадает в кандидаты для AI-анализа. При наличии релевантного контекста ссылка появляется в "Полезные материалы" preview.

---

## TC-LNK-014: Ссылки в preview не разворачиваются в веб-превью Telegram

**Предусловия:**
- Preview показан, есть "Полезные материалы"

**Шаги:**
1. Получить preview-сообщение

**Ожидаемый результат:** Telegram не разворачивает веб-превью ни для одной ссылки. Формат каждой ссылки: "• {role}: {url} — {reason}".
