# Сьют 4: /start_discussion - начало обсуждения

---

## TC-SD-001: /start_discussion без установленного проекта

**Предусловия:**
- Проект Todoist не установлен
- Todoist API доступен

**Шаги:**
1. Отправить `/start_discussion`
2. Отправить произвольное текстовое сообщение

**Ожидаемый результат:** Бот отвечает "Сначала выберите проект Todoist:" с inline-списком проектов. Сессия не создаётся - отправленное сообщение не сохраняется в сессию.

---

## TC-SD-002: /start_discussion с установленным проектом

**Предусловия:**
- Проект Todoist установлен
- Нет активной сессии

**Шаги:**
1. Отправить `/start_discussion`

**Ожидаемый результат:** Бот отвечает "Обсуждение началось." и сообщает, что сообщения будут сохраняться до `/create_task` или `/cancel`. Активная сессия создана, owner_id = ID отправителя.

---

## TC-SD-003: /start_discussion при уже активной сессии

**Предусловия:**
- Сессия уже запущена

**Шаги:**
1. Отправить `/start_discussion` повторно

**Ожидаемый результат:** Бот отвечает "Обсуждение уже идёт! Прежде, чем начать новое завершите текущее." Старая сессия не перезаписана.

---

## TC-SD-004: Владелец сессии - пользователь, отправивший команду

**Предусловия:**
- Групповой чат с Пользователями A и B
- Проект установлен

**Шаги:**
1. Пользователь A отправляет `/start_discussion`
2. Пользователь B пишет несколько сообщений
3. Пользователь B отправляет `/create_task`
4. Пользователь A отправляет `/create_task`

**Ожидаемый результат:** На шаге 3 бот отвечает "Только автор обсуждения может создать задачу по итогам обсуждения." На шаге 4 AI анализирует обсуждение и показывает preview.

---

## TC-SD-005: После закрытой сессии можно начать новую

**Предусловия:**
- Предыдущая сессия завершена через `/cancel` - "Завершить"

**Шаги:**
1. Отправить `/start_discussion`

**Ожидаемый результат:** Бот отвечает "Обсуждение началось." Новая сессия создана без ошибок.

---

## TC-SD-006: Кнопка "Начать обсуждение" эквивалентна команде

**Предусловия:**
- Проект установлен, нет активной сессии
- Главная клавиатура видна

**Шаги:**
1. Нажать "Начать обсуждение"

**Ожидаемый результат:** Поведение идентично прямой команде `/start_discussion`.
