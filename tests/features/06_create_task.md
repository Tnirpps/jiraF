# Сьют 6: /create_task - инициация черновика задачи

---

## TC-CT-001: /create_task без установленного проекта

**Предусловия:**
- Проект не установлен
- Todoist API доступен

**Шаги:**
1. Отправить `/create_task`

**Ожидаемый результат:** Бот отвечает "Сначала выберите проект Todoist:" с inline-списком проектов.

---

## TC-CT-002: /create_task без активной сессии

**Предусловия:**
- Проект установлен
- Нет активной сессии

**Шаги:**
1. Отправить `/create_task`

**Ожидаемый результат:** Бот отвечает "Нет активного обсуждения. Начните его командой /start_discussion."

---

## TC-CT-003: /create_task в сессии без сообщений

**Предусловия:**
- Проект установлен
- Сессия запущена, но ни одного текстового сообщения не написано

**Шаги:**
1. Отправить `/create_task`

**Ожидаемый результат:** Бот отвечает "В обсуждении нет сообщений, чтобы создать задачу." Сессия остаётся открытой.

---

## TC-CT-004: Не-владелец вызывает /create_task

**Предусловия:**
- Групповой чат, Пользователь A запустил сессию

**Шаги:**
1. Пользователь B отправляет `/create_task`

**Ожидаемый результат:** Бот отвечает "Только автор обсуждения может создать задачу по итогам обсуждения." AI-запрос не выполняется. Сессия не изменилась.

---

## TC-CT-005: Успешный вызов - черновик создан и preview показан

**Предусловия:**
- Проект установлен, активная сессия с 3+ сообщениями
- AI API доступен

**Шаги:**
1. Отправить `/create_task`

**Ожидаемый результат:** Появляется preview с текстом "Черновик задачи готов." Поля "Название", "Описание", "Тип задачи" и "Приоритет" присутствуют и заполнены. Под preview три inline-кнопки: "Подтвердить", "Редактировать", "Отменить создание".

---

## TC-CT-006: Preview содержит секцию "Полезные материалы" при наличии URL

**Предусловия:**
- Сессия содержит сообщения с URL

**Шаги:**
1. Отправить `/create_task`

**Ожидаемый результат:** Preview содержит секцию "Полезные материалы" с ссылками в формате "* {role}: {url} - {reason}".

---

## TC-CT-007: Секция "Полезные материалы" отсутствует, если URL не было

**Предусловия:**
- Сессия содержит только текстовые сообщения без URL

**Шаги:**
1. Отправить `/create_task`

**Ожидаемый результат:** В preview нет секции "Полезные материалы".

---

## TC-CT-008: Preview показывает "Можно ещё уточнить" при незаполненных полях шаблона

**Предусловия:**
- Сессия с кратким обсуждением, не все поля шаблона заполнены AI

**Шаги:**
1. Отправить `/create_task`

**Ожидаемый результат:** Preview содержит текст "Можно ещё уточнить: похоже, перед созданием задачи стоит обсудить {поля}." и подсказку "Если хочешь, нажми Редактировать и дополни это в задаче."

## TC-CT-011: AI вернул приоритет вне диапазона 1-4 - нормализация до 1

**Предусловия:**

**Шаги:**
1. Отправить `/create_task`

## TC-CT-012: AI недоступен

**Предусловия:**
- AI API недоступен
- Сессия с сообщениями активна

**Шаги:**
1. Отправить `/create_task`

**Ожидаемый результат:** Бот отвечает "AI суммаризация не удалась(. Попробуйте заново". Сессия остаётся открытой.

---

## TC-CT-013: Повторный /create_task после "Отменить создание"

**Предусловия:**
- Preview был показан, пользователь нажал "Отменить создание"
- Сессия остаётся открытой

**Шаги:**
1. Написать ещё одно сообщение в сессию
2. Отправить `/create_task`

**Ожидаемый результат:** AI анализирует все сообщения, включая новое. Показывается новый preview. Предыдущий черновик не используется.

---

## TC-CT-014: Кнопка "Создать задачу" эквивалентна команде /create_task

**Предусловия:**
- Проект установлен, активная сессия с сообщениями

**Шаги:**
1. Нажать "Создать задачу"

**Ожидаемый результат:** Поведение идентично прямой команде `/create_task`.

---
