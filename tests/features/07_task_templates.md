# Сьют 7: Типы задач и шаблонные поля в preview

---

## TC-TMPL-001: Тип "bug" - поля баг-репорта в preview

**Предусловия:**
- Сессия содержит обсуждение о сломанном поведении системы
- AI классифицировал как `bug`

**Шаги:**
1. Отправить `/create_task`

**Ожидаемый результат:** Preview содержит "Тип задачи: Баг". Заполненные AI поля ("Что сломано", "Шаги воспроизведения", "Ожидаемое поведение", "Фактическое поведение") присутствуют. Незаполненные поля попадают в секцию "Можно ещё уточнить".
