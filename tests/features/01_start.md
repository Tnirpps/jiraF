# Сьют 1: /start - онбординг

---

## TC-ST-001: Первый запуск - проект не установлен

**Предусловия:**
- Бот никогда не использовался в этом чате
- Проект Todoist не установлен
- Todoist API доступен и возвращает хотя бы один проект

**Шаги:**
1. Открыть чат с ботом
2. Отправить `/start`

**Ожидаемый результат:** Бот отвечает приветственным текстом с описанием возможностей. Под сообщением показан inline-список проектов Todoist: каждый проект - отдельная кнопка-строка.

---

## TC-ST-002: Повторный /start - проект уже установлен

**Предусловия:**
- Проект Todoist установлен командой `/set_project`
- Нет активной сессии

**Шаги:**
1. Отправить `/start`

**Ожидаемый результат:** Бот отвечает приветственным текстом. Inline-кнопки с проектами отсутствуют. Показана reply-клавиатура с 6 кнопками.

---

## TC-ST-003: /start в групповом чате

**Предусловия:**
- Бот добавлен в групповой чат

**Шаги:**
1. Отправить `/start` в группе

**Ожидаемый результат:** Бот отвечает без ошибок. Текст приветствия корректный, клавиатура или inline-список проектов присутствует.

---

## TC-ST-004: /start при недоступном Todoist (проект не установлен)

**Предусловия:**
- Проект не установлен
- Todoist API недоступен

**Шаги:**
1. Отправить `/start`

**Ожидаемый результат:** Бот отвечает сообщением, содержащим "Не удалось загрузить проекты Todoist:". Бот не падает и продолжает отвечать на следующие сообщения.

---

## TC-ST-005: Главная клавиатура содержит все 6 кнопок

**Предусловия:**
- Проект установлен

**Шаги:**
1. Отправить `/start`

**Ожидаемый результат:** Reply-клавиатура содержит три ряда. Первый ряд: "Выбрать проект" и "Начать обсуждение". Второй ряд: "Создать задачу" и "Завершить обсуждение". Третий ряд: "Список задач" и "Помощь".
